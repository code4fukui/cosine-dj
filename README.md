# CosineDJ

CosineDJ is a browser-based drum machine built using the Tone.js library.

## Demo
You can try the live demo at https://code4fukui.github.io/cosine-dj/

## Features
- Customizable drum patterns with on/off checkboxes
- Low-pass filter and distortion controls
- Automatic filter modulation
- Supports Kick, Hihat, and Snare drum sounds

## Usage
1. Click the "play" button to start the drum pattern.
2. Adjust the low-pass filter and distortion using the sliders.
3. Enable the "filter auto" checkbox for automatic filter modulation.
4. Click the "stop" button to stop the pattern.
