# CosineDJ

CosineDJは、ブラウザベースのWebオーディオアプリです。ビートを生成し、フィルターを適用することで、簡単に音楽を作成できます。

## デモ
[CosineDJ](https://code4fukui.github.io/cosine-dj/)で実際に試してみることができます。

## 機能
- ビートパターンの生成と再生
- ローパスフィルター
- ディストーションフィルター
- 自動フィルター変更

## 必要環境
- 任意のWebブラウザ

## 使い方
1. ページを開くと、3つのビートパターンが表示されます。
2. パターンの各ボックスをクリックして、ビートをオン/オフにできます。
3. 「play」ボタンを押すと、ビートが再生されます。
4. フィルターのスライダーを操作して、サウンドを変更できます。
5. 「auto」チェックボックスをオンにすると、フィルターが自動的に変化します。
6. 「stop」ボタンを押すと、ビートの再生が停止します。
